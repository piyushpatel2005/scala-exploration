# Scala Exploration

Scala is a general-purpose programming language that is designed to express common programming patterns in a concise, elegant, and type-safe way. It smoothly integrates features of object-oriented and functional languages. It is also fully interoperable with Java.

The initial lessons can be tested with the Scala REPL or Scala workbook. The later lessons will require a full Scala project setup. The code for the lessons can also be found in this repository under [core module](../../core/).
